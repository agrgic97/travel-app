# Travel App – NativeScript Vue 3 Starter

Diese Beispiel-App ist ein einfacher Einstieg, um **NativeScript mit Vue 3** kennenzulernen.

## Was die App kann

- Startet eine Vue-3-basierte NativeScript-App.
- Zeigt eine Seite mit Begrüßungstext.
- Enthält einen Button, der einen Counter erhöht.
- Nutzt `ref` und `computed` aus der Vue Composition API.

## Voraussetzungen

- Node.js 20+
- NativeScript CLI (`npm i -g nativescript`)
- Android Studio und/oder Xcode (für Emulator oder Gerät)

## Installation

```bash
npm install
```

## Starten

Android:

```bash
npm run start
```

iOS:

```bash
npm run start:ios
```

## TypeScript/Vue Check

```bash
npm run typecheck
```
